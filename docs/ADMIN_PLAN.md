# Rencana Implementasi Admin Dashboard & Supabase Integration

## 1. Persiapan Supabase (Database)
Anda perlu membuat project baru di [supabase.com](https://supabase.com) dan mendapatkan `URL` serta `ANON_KEY`.

### Schema Database (Dibuat di SQL Editor Supabase)

#### Table: `news` (Berita)
```sql
create table news (
  id bigint generated by default as identity primary key,
  title text not null,
  slug text unique not null,
  date date not null,
  image_url text,
  content text,
  excerpt text,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);
```

#### Table: `events` (Kegiatan)
```sql
create table events (
  id bigint generated by default as identity primary key,
  title text not null,
  slug text unique not null,
  date date not null,
  location text,
  image_url text,
  description text,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);
```

#### Storage Buckets
Anda perlu membuat Storage Bucket di Supabase bernama:
- `images` (Public)

## 2. Struktur Folder Admin
Admin akan terpisah dari layout publik website utama.
- `src/app/(admin)/admin/` -> Base URL untuk admin.
- `src/app/(admin)/admin/dashboard/` -> Halaman utama dashboard.
- `src/app/(admin)/admin/news/` -> CRUD Berita.
- `src/app/(admin)/admin/events/` -> CRUD Event.

## 3. Langkah Implementasi
1. Install Library: `@supabase/supabase-js`
2. Setup Environment Variables (`.env.local`).
3. Buat Supabase Client Helper.
4. Buat Halaman Login Admin.
5. Buat Layout Admin Dashboard (Sidebar & Header).
6. Implementasi CRUD (Create, Read, Update, Delete) untuk konten.

## 4. Keamanan
Untuk tahap awal, kita akan menggunakan *Magic Link* atau *Email/Password* auth bawaan Supabase untuk login admin.
